Advanced XML sitemap generator with auto-updates, category grouping, manual override, search engine pinging, and full in-admin view & edit.

== Description ==

Big SEO Sitemap is a powerful WordPress plugin that automatically generates and manages your XML sitemap.

Features:
* Auto-generates sitemap on activation
* Refreshes every 24 hours via WP Cron
* Groups posts by category
* Priority & change frequency control per content type
* Manual regeneration from the dashboard
* Pings Google & Bing after every update
* Full in-admin view, edit, and exclude per URL
* Raw XML editor

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/big-seo-sitemap/`
2. Activate the plugin through the 'Plugins' screen
3. Go to Big SEO Sitemap in the admin menu
